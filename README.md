# Cinema Booking — Дипломная работа «Frontend-разработчик» (Нетология)

Сайт онлайн-бронирования билетов в кинотеатр + административная панель.

## О проекте

Проект представляет собой полноценное SPA-приложение для бронирования билетов в кинотеатр.  
Реализованы две основные части:
- **Гостевая часть** — просмотр афиши, выбор сеанса, выбор мест, бронирование билета с генерацией QR-кода.
- **Административная часть** — управление залами, фильмами, сеансами и ценами.
